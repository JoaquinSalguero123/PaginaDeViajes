--Subir archivos a GitHub (desde VS Code o terminal)

Abrir la carpeta del proyecto en Visual Studio Code.

1)Inicializar Git en tu proyecto:

git init


2)Conectar tu proyecto a GitHub (usando la URL de tu repo):

git remote add origin https://github.com/usuario/mi-repo.git


3)Agregar archivos al seguimiento:

git add .


4)Crear un commit con los cambios:

git commit -m "Subo mi proyecto"


5)Subir los archivos a GitHub:

git branch -M main
git push -u origin main
